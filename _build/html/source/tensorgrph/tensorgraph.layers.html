class tensorgraph.layers.misc.Slim(*args, **kwargs)¶

Bases: tensorgraph.layers.template.BaseModel

Slim converts Tensorgraph Sequential Mode into Functional Mode. Similar to Keras Sequential and Functional.

Example how to use Slim class SampleModel(BaseModel):

@BaseModel.init_name_scope def __init__(self):

self.startnode = StartNode([None]) conv1 = Slim([self.startnode], [Conv2D(num_filters=16, kernel_size=(3,3), stride=(1,1), padding=’SAME’)]) flat = Slim([conv1.endnode], [Flatten()]) batch = Slim([flat.endnode], [BatchNormalization()]) l1 = Slim([batch.endnode], [Linear(this_dim=30) ]) l2 = Slim([l1.endnode], [Linear(this_dim=10) ]) l3 = Slim([l2.endnode], [Linear(this_dim=5) ]) self.endnode = EndNode(prev=[l3.endnode])
