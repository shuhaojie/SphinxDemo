tensorgraph.dataset.mnist.read_mnist_images(fn, dtype=None)¶

Read MNIST images from the original ubyte file format. If the dtype provided was boolean, the resulting array will be boolean with True if the corresponding pixel had a value greater than or equal to 128, False otherwise. If the dtype provided was a float or complex dtype, the values will be mapped to the unit interval [0, 1], with pixel values that were 255 in the original unsigned byte representation equal to 1.0.

Parameters

  • fn (str or object) – Filename/path from which to read labels, or an open file object for the same (will not be closed for you).

  • dtype (str or object, optional) – A NumPy dtype or string that can be converted to one. If unspecified, images will be returned in their original unsigned byte format.

Returns

An image array, with individual examples indexed along the first axis and the image dimensions along the second and third axis.

Return type

images (ndarray, shape (n_images, n_rows, n_cols))

Parameters

  • transform_matrix – Numpy array specifying the geometric transformation.

  • fill_mode – Points outside the boundaries of the input are filled according to the given mode (one of {‘constant’, ‘nearest’, ‘reflect’, ‘wrap’}).

  • cval – Value used for points outside the boundaries of the input if mode=’constant’.
